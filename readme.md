# Java Samples for Lecture "Techniken der Programmentwicklung"

This repository contains the sample source code used in my lecture on advanced Java programming at the University of Applied Sciences in Mannheim. The repository is primarily for students interested in the Java code used during the lecture. For other people, it may be of limited use.
